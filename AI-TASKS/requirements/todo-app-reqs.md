The TODO-APP is a mobile application that allows users to create, edit, and delete todo-tasks.
A todo-task has a numerical unique id, a text description, a state (open, finished), a creation timestamp and a due date.

The UI design of the tofo-app is shown in the following screenshot: ./todo-app-design.png

The design has a title bar at the top, an input field with an add button at the middle and a list of todo-tasks at the bottom.
Adding a new todo-task is done by entering a text in the input field and clicking the add button or by pressing the enter-key.
The todo-list displays the state as a checkbox, the description, the due date and a trash-can icon.
Clicking the checkbox toggles the state of the todo-task.
Clicking the trash-can icon deletes the todo-task.
