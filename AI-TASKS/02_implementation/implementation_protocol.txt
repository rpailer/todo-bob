Setp 4) when running the app with 'npm run dev', the systems runs out of memory.

Analyse and fix the code wothout starting it.

Fixed the infinite loop causing memory exhaustion. The issue was in [`hooks/useTodos.ts`](hooks/useTodos.ts:126-129) where the `useEffect` had `fetchTodos` in its dependency array, causing it to re-run continuously.

**Root cause:** The `useEffect` was triggering on every render because `fetchTodos` was being recreated, even though it had an empty dependency array in its `useCallback`.

**Solution:** Changed the `useEffect` dependency array from `[fetchTodos]` to `[]` to ensure it only runs once on mount, preventing the infinite loop.

The app should now run normally with `npm run dev` without memory issues.
